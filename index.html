Reference sheet HTML&CSS

Semantic HTML

Tag Name Description Attributes
<a> Anchor Element to make link to external source, other/part of page (open/close) href="" for source, target="_blank" to open in new tab, elements between open and closing tags will be clickable link, in <li> the <a> must be inside the tags
<img> Image Element to insert a picture from files or external source (selfclose) src="" for source, alt="" for description of picture (mostly for search engine and screenreader)
<audio> Audio Element to insert an audio from files or external source (open/close) src="" for source, controls for play/pause/volume, autoplay to start audio on page load
<video> Video Element to insert a video from files or external source (open/close) src="" for source, include "Video not supported" between open and closing tags in case video can't be loaded, controls for play/pause/volume, autoplay to start video on page load
<embed> Embed Element to insert any type of media (photo, video, GIF etc.) (selfclose) THIS TAG IS ADVISED NOT TO BE USED ANYMORE! Taught for legacy purpose only
<figure> Figure Element to enclose media (photo, video etc.) (open/close) <figcaption>(open/close) to add caption/description to be grouped with the media

CSS Stuffies

Name Description Properties
overflow Sets the desired behavior for when an element's content is too big to fit in its block formatting context (default) visible = content may be rendered outside the container, hidden = content is cut off and hidden ,scroll = scrollbar added to view all content
visibility Sets the visibility of an element without changing the layout of the page hidden = hides the element but the space of the element will still be reserved on the page, visible = displays the element ,collapse = collapses the element (ea. in tables this "removes" the row as display: none, on other elements it works as hidden)
box-sizing Controls how an elements height and width are rendered (in/excluding padding & border) (default) content-box = content will be rendered the set height and width with the padding and border being added on top, border-box = content will be rendered to a calculated height and width after padding and border are taken off the set height and width
